اضافه کردن ردیف

ردیف های جدید می توانند به راحتی با استفاده از روش API (row.add ()) به جدول داده اضافه شوند. به سادگی با عملکرد API با داده هایی که برای ردیف جدید مورد استفاده قرار می گیرند، بایستی آن آرایه یا شی باشد. ردیف های چندگانه را می توان با استفاده از روش (rows.add ()) (توجه داشته باشید تعداد). داده ها نیز می توانند با داده های (row (). data () و row () () () حذف شوند.)

جستجو ستون فردی (ورودی را انتخاب کنید)

این مثال تقریبا یکسان است به عنوان نمونه جداگانه متن مبتنی بر متن و عملکرد مشابهی را فراهم می کند، اما در این مورد با استفاده از کنترل های انتخاب ورودی. پس از جدول اولیه، API برای ساختن ورودی های انتخاب شده از طریق استفاده از روش ستون () data () برای دریافت داده ها برای هر ستون به نوبه خود استفاده می شود. روش های helper منحصر به فرد () و مرتب سازی () نیز برای کاهش داده ها برای تنظیم ورودی به عناصر منحصر به فرد و دستورالعمل استفاده می شود. در نهایت، رویداد تغییر از ورودی انتخاب شده برای نشان دادن جستجوی ستون با استفاده از روش column (). search () استفاده می شود.
