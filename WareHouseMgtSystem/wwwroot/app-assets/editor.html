ویرایشگر کویل

ویرایشگر کویل با زاویه و TypeScript. ngx-quill جدید زاویه ای 2 و فراتر از اجرای ngQuill است.

ویرایشگر پیش فرض

حباب ویرایشگر

Bubbleیک تم مبتنی بر ابزار راهنمای ساده است. دوبار کلیک بر روی متن را امتحان کنید. برای ویرایش باید حباب را با نوار ابزار ببینید.

کویل ویراستار متن


کویل یک آزاد است، متن باز ویرایشگر WYSIWYG برای وب مدرن ساخته شده است. با آنمعماری مدولار و بیانگررابط کاربری، آن را کاملا قابل تنظیم است به هر نیاز.


شروع کار آسان است


// <link href="https://cdn.quilljs.com/1.0.5/quill.snow.css" rel="stylesheet">
	// <script src="https://cdn.quilljs.com/1.0.5/quill.min.js" type="text/javascript"></script>
	var quill = new Quill('#editor', {
	modules: {
		toolbar: '#toolbar'
	},
	theme: 'snow'
	});
	// Open your browser's developer console to try out the API!

ویرایشگر کامل

به طور پیش فرض تمام فرمت ها فعال شده اند و اجازه می دهند در یک ویرایشگر Quill موجود باشند و می توانند با آن پیکربندی شوندformats گزینه. این جدا از اضافه کردن یک کنترل درToolbar. به عنوان مثال، شما می توانید Quill را مجددا راه اندازی کنید تا محتوای جسورانه را به یک ویرایشگر که هیچ دکمه bold در نوار ابزار ندارد، جاگذاری کنید.
