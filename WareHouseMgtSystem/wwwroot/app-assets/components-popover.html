موقعیت اعلان

چهار گزینه در دسترس هستند: بالا، راست، پایین، و سمت چپ تراز.

اعلان بالا پایه

اضافه کردن کلاسdata-toggle="popover"و data-placement="top" برای اضافه کردن اعلان بالا

اعلان راست پایه

اضافه کردن کلاسdata-toggle="popover" وdata-placement="right" برای اضافه کردن اعلان راست

اعلان پایه پایین

اضافه کردنdata-toggle="popover" وdata-placement="bottom"برای اضافه کردن اعلان پایین

اعلان پایه چپ

اضافه کردنdata-toggle="popover"وdata-placement="left"اضافه کردن اعلان چپ

راه اندازی اعلان

اعلان با استفاده از - کلیک| شناور | تمرکز | | گزینه های دستی. شما ممکن است چند عامل را بردارید آنها را با فضایی جدا کنید. "کتابچه راهنمای" نمیتوان با هر ماژول دیگری ترکیب کرد.

شناور

استفاده ازdata-trigger="hover"برای ماشه شناور

کلیک

استفاد ازdata-trigger="click"برای کلیک اعلان. این یک اعلان پیش فرض است

تمرکز

استفاده از data-trigger="focus" برای تمرکز در راهنما

کتابچه راهنمای

استفاده ازdata-trigger="manual"برای راهنما دستی شما می توانید با استفاده از js نمایش / پنهان کنید

گزینه راهنما

غیر فعال کردن انیمیشن

استفاده ازdata-animation="false" برای حذف انیمیشن محو شدن. پیش فرض درست است

تأخیر نمایش / مخفی کردن

تاخیر در نمایش و پنهان کردن راهنما(ms)- به نوع ماشه دستی اعمال نمی شود.

پشتیبانی از HTML

استفاده ازdata-html="true"برای راهنما پشتیبانی HTML.

الگو راهنما

پایه HTML برای ایجاد راهنما استفاده می شود. عنوان راهنما به داخل تزریق خواهد شد.popover-inner. .arrowعنصر wrapper outermost خواهد بود که باید کلاس داشته باشد.popover.

روش راهنما

این در نظر گرفته شده است "دستی" راه اندازی popover.Popovers که هر دو عنوان و محتوای صفر هستند هرگز نشان داده نمی شود.

نمایش

آشکار یک راهنما عنصر. قبل از اینکه popover در واقع نشان داده شود، به تماس گیرنده برگشت می کند (i.e. قبل ازshown.bs.popover رویداد رخ می دهد)

پنهان

پنهان کردن عنصر popover. قبل از اینکه راهنما در واقع پنهان شده باشد (قبل از آن، به caller بازگردد)hidden.bs.popover رویداد رخ می دهد)

تغییر وضعیت

تغییر ظاهر عنصر قبل از اینکه راهنما در واقع نشان داده شده یا پنهان شده باشد (قبل ازshown.bs.popover یا hidden.bs.popover رویداد رخ می دهد).

انتقال

پنهان و پنهان یک عنصر را پنهان می کند. راهنما هایی که از اعلان استفاده می کنند نمی توانند به طور جداگانه بر روی عناصر ماشه نسل نابود شوند.

نمایش رویداد

این رویداد بلافاصله پس از آتش بسshowروش نمونه به نام.

رویداد نشان داده شده

این رویداد هنگامی که راهنما برای کاربر قابل مشاهده است، اخراج می شود (منتظر انتقال CSS برای تکمیل است).

رویداد پنهان
